# 🌻 我的花园世界

一款像素风格的花园经营游戏！

## 🎮 游戏特色

### 像素风格
- ✅ 复古8-bit像素视觉
- ✅ 硬边阴影和阶梯动画
- ✅ 像素字体和UI
- ✅ 方块风格的植物和场景

### 完整玩法
- ✅ **10种植物**: 从郁金香到彩虹花
- ✅ **4种工具**: 浇水、种植、收获、清除
- ✅ **经营系统**: 购买种子 → 种植 → 浇水 → 收获 → 赚钱
- ✅ **升级系统**: 等级、经验、解锁新植物
- ✅ **体力系统**: 每天恢复，限制操作
- ✅ **天数系统**: 植物生长需要时间
- ✅ **背包系统**: 存储收获的植物
- ✅ **统计系统**: 完整的游戏数据

## 🎯 如何游戏

### 基本操作
1. **选择工具**: 点击底部工具栏选择要使用的工具
2. **点击格子**: 对花园格子使用工具
3. **下一天**: 点击"下一天"让植物生长，恢复体力
4. **购买种子**: 在左侧商店选择要种植的植物

### 工具说明
- 💧 **浇水**: 点击格子 → 耕地或浇水（需要先耕地才能种）
- 🌱 **种植**: 先选种子，再点击已耕地的格子种下
- ✂️ **收获**: 点击成熟的植物（会发光）收获
- 🪓 **清除**: 清除植物或恢复草地

### 游戏循环
1. 耕地 → 播种 → 浇水
2. 等待生长（需要几天，每天都要浇水！）
3. 收获 → 卖钱 → 买更多种子
4. 升级 → 解锁新植物 → 扩大种植

## 📊 游戏系统

### 等级与解锁
- **3级**: 解锁玫瑰、玉米
- **5级**: 解锁茄子
- **7级**: 解锁草莓
- **10级**: 解锁西瓜
- **15级**: 解锁樱桃
- **20级**: 解锁彩虹花

### 体力消耗
- 耕地: 2点
- 浇水: 3点
- 种植: 5点
- 收获: 8点
- 清除: 2点

### 保存游戏
游戏会自动保存，也可以点击"💾 保存"按钮手动保存！
