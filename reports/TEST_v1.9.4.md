## 🔧 Hotfix 修复（2026-03-12 09:50）

### 问题发现
用户反馈"画布还是没有变大"

### 根本原因
**GameCanvas 组件**移动端固定使用 `blockSize = 20px`：
```typescript
// 修复前
const actualBlockSize = blockSize ?? (window.innerWidth < 768 ? 20 : 30);
// 画布实际大小：10 列 × 20px = 200px（即使容器是 360px）
```

### 修复内容
- **文件**: `src/components/game/GameCanvas.tsx`
- **修改**: 移动端 blockSize 从 20px 提升到 30px
- **结果**: 画布实际大小从 200px 增大到 300px（10 列 × 30px）

### 修复后尺寸
| 组件 | 修复前 | 修复后 | 提升 |
|------|--------|--------|------|
| ResponsiveLayout 容器 | 500px | 500px | - |
| 游戏画布容器 | 360px | 360px | - |
| **GameCanvas 实际大小** | **200px** | **300px** | **+50%** |
