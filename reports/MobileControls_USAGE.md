# 虚拟按键使用指南

**版本**: v1.9.1  
**最后更新**: 2026-03-11

---

## 📱 快速开始

### 1. 打开设置

在游戏主界面，点击 **📱 按键** 按钮打开虚拟按键设置面板。

### 2. 启用虚拟按键

在设置面板中：
1. 点击 **启用虚拟按键** 切换按钮
2. 状态变为 **已启用**（绿色）表示开启
3. 状态变为 **已禁用**（红色）表示关闭

### 3. 调整设置

- **按键尺寸**: 选择小/中/大
- **透明度**: 拖动滑块调节（50%-100%）
- **实时预览**: 查看当前设置效果

### 4. 保存设置

设置会自动保存到浏览器，下次打开游戏时自动恢复。

## 🎮 按键布局

```
┌─────────────────────────┐
│   滑动控制 | 双击硬降    │ ← 触摸提示区
├─────────────────────────┤
│  [⬅️]   [🔄]   [➡️]    │ ← 方向控制
│  [⬇️ 软降] [💥 硬降]    │ ← 降落控制
│  [⏸️ 暂停] [🔄 重开]    │ ← 功能按钮
└─────────────────────────┘
```

### 按键功能

| 按键 | 功能 | 说明 |
|------|------|------|
| ⬅️ | 左移 | 方块向左移动，长按连发 |
| 🔄 | 旋转 | 方块顺时针旋转 |
| ➡️ | 右移 | 方块向右移动，长按连发 |
| ⬇️ 软降 | 软降落 | 方块加速下落 |
| 💥 硬降 | 硬降落 | 方块直接落到底部 |
| ⏸️ 暂停 | 暂停游戏 | 暂停/继续游戏 |
| 🔄 重开 | 重新开始 | 重新开始当前游戏 |

## 🖐️ 触摸手势

即使启用了虚拟按键，触摸手势仍然可用：

### 滑动控制
- **左滑**: 方块左移
- **右滑**: 方块右移
- **下滑**: 软降落

### 双击硬降
- **快速双击** 触摸区域：硬降落

### 使用建议
- 习惯触摸手势的用户可以关闭虚拟按键
- 需要精确控制的用户建议使用虚拟按键
- 两者可以同时使用，互不影响

## ⚙️ 设置说明

### 按键尺寸

| 尺寸 | 按钮大小 | 适用场景 |
|------|----------|----------|
| 小 | 40px × 40px | 小屏幕手机，单手操作 |
| 中 | 50px × 50px | 默认推荐，大多数设备 |
| 大 | 60px × 60px | 大屏幕手机/平板，双手操作 |

### 透明度

- **50%-70%**: 半透明，减少遮挡
- **80%-90%**: 默认，平衡可见性和美观
- **100%**: 完全不透明，最高可见度

### 重置设置

点击 **🔄 重置为默认值** 恢复默认设置：
- 启用状态：已启用
- 尺寸：中
- 透明度：90%

## 🎨 颜色说明

虚拟按键使用颜色编码帮助识别：

- **🔵 青色** (#00ffff): 水平移动（左/右）
- **🔴 粉色** (#ff00ff): 旋转/重开
- **🟢 绿色** (#00ff80): 软降落
- **🟠 橙色** (#ffa600): 硬降落
- **🔴 红色** (#ff4444): 暂停

## 💡 使用技巧

### 1. 长按连发
- 按住 **⬅️** 或 **➡️** 不放
- 方块会连续移动
- 适合快速调整位置

### 2. 防误触保护
- 游戏开始后 500ms 内按键禁用
- 防止开始游戏时误触
- 之后自动启用

### 3. 横屏模式
- 旋转手机自动切换横屏布局
- 按键自动调整为横向排列
- 触摸提示区自动隐藏

### 4. 游戏结束
- 游戏结束时按键自动禁用
- 防止无效操作
- 重新开始后自动启用

## 🔧 故障排除

### 问题：按键无响应

**可能原因**:
1. 虚拟按键未启用
2. 游戏处于暂停状态
3. 游戏已结束

**解决方法**:
1. 检查设置中是否已启用
2. 继续游戏
3. 重新开始游戏

### 问题：按键位置不对

**可能原因**:
1. 屏幕尺寸变化
2. 浏览器缩放

**解决方法**:
1. 刷新页面
2. 重置浏览器缩放（100%）
3. 调整按键尺寸设置

### 问题：设置不保存

**可能原因**:
1. 浏览器禁止 localStorage
2. 隐私模式/无痕模式

**解决方法**:
1. 允许网站存储数据
2. 使用正常浏览模式
3. 清除浏览器缓存后重试

## 📊 性能影响

虚拟按键组件经过优化，对性能影响极小：

- **代码大小**: ~8KB（压缩后）
- **内存占用**: < 1MB
- **渲染开销**: 可忽略
- **帧率影响**: < 1fps

可以放心使用，不会影响游戏流畅度。

## 🔒 隐私说明

虚拟按键设置存储在本地浏览器：

- **存储位置**: localStorage
- **存储内容**: 启用状态、尺寸、透明度
- **数据范围**: 仅本网站
- **隐私影响**: 无，不收集个人信息
