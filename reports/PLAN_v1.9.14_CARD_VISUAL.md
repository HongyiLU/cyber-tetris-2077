# v1.9.14 卡组卡牌视觉优化计划

**版本**: v1.9.14

**状态**: 需求分析

## 📋 需求概述

将卡组编辑器中的方块显示修改为完整的卡牌形式，包含卡名、卡面、卡牌效果和稀有度系统。

---

## 🎯 功能需求

### 1. 卡牌视觉元素

每张卡牌需要包含以下元素：

| 元素 | 说明 | 位置 |
|------|------|------|
| **卡名** | 方块名称（如"I-方块"、"T-方块"） | 卡牌顶部 |
| **卡面** | 方块的实际形状显示 | 卡牌中央 |
| **卡牌效果** | 简短描述方块特性 | 卡牌底部 |
| **稀有度** | 白/绿/蓝/紫/橙 五种稀有度 | 边框颜色 + 背景光效 |

### 2. 稀有度系统

| 稀有度 | 颜色 | 边框 | 背景光效 | 分布 |
|--------|------|------|----------|------|
| ⚪ 普通 | 白色 (#ffffff) | 白色边框 | 微弱白光 | 常见 |
| 🟢 罕见 | 绿色 (#00ff00) | 绿色边框 | 绿色光晕 | 较常见 |
| 🔵 稀有 | 蓝色 (#0088ff) | 蓝色边框 | 蓝色光晕 | 中等 |
| 🟣 史诗 | 紫色 (#a855f7) | 紫色边框 | 紫色光晕 | 较稀有 |
| 🟠 传说 | 橙色 (#ff8800) | 橙色边框 | 橙色光晕 + 粒子 | 最稀有 |

### 3. 卡牌效果描述

为每种方块设计独特的效果描述：

| 方块 | 卡名示例 | 效果描述 | 稀有度 |
|------|---------|---------|--------|
| I | 直线冲击 | 消除整行，造成穿透伤害 | 🟣 史诗 |
| O | 坚固壁垒 | 稳定下落，不易被打断 | ⚪ 普通 |
| T | 旋转突击 | 可旋转，灵活应对 | 🟢 罕见 |
| S | 曲折前进 | 连续消除获得连击加成 | 🔵 稀有 |
| Z | 反向突袭 | 反向移动时速度提升 | 🔵 稀有 |
| L | 角落专家 | 适合填充角落空隙 | 🟢 罕见 |
| J | 镜像战士 | 与 L 方块对称，效果相同 | 🟢 罕见 |

## 🎨 设计要求

### 卡牌尺寸
- 宽度：120px
- 高度：180px
- 圆角：8px
- 厚度感：阴影 + 渐变

### 卡牌布局
```
┌─────────────────┐
│  [稀有度图标]    │ ← 顶部：稀有度徽章
│  I-方块         │ ← 卡名
├─────────────────┤
│                 │
│    ████████     │ ← 卡面：方块形状
│                 │
├─────────────────┤
│ 消除整行，造成  │ ← 效果描述
│ 穿透伤害        │
└─────────────────┘
```
