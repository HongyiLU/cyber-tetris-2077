## 🎯 需求分析

### 用户反馈
> "现在选择好敌人确定后会直接开始游戏，希望添加一个 3 秒倒计时，倒计时结束后才开始游戏。并且在游戏结束后，弹出弹窗提示玩家要不要回到标题页，或是重新挑战。"

### 核心需求
1. **3 秒倒计时** - 选择敌人后显示倒计时，结束后才开始游戏
2. **游戏结束弹窗** - 游戏结束后提供"回到标题页"和"重新挑战"选项

### 功能范围
- ✅ 敌人选择界面添加倒计时显示
- ✅ 倒计时 3 秒后自动开始游戏
- ✅ 游戏结束弹窗（回到标题页/重新挑战）
- ✅ 倒计时可取消（返回按钮）
- ✅ 移动端和桌面端适配

### 不在范围
- ❌ 不修改现有游戏逻辑
- ❌ 不修改敌人选择逻辑

### 数据结构
```typescript
// 倒计时组件 props
interface GameStartCountdownProps {
  seconds: number;        // 倒计时秒数（默认 3）
  onComplete: () => void; // 倒计时完成回调
  onCancel?: () => void;  // 取消回调
}

// 游戏结束弹窗 props
interface GameEndModalProps {
  score: number;          // 得分
  lines: number;          // 消除行数
  onRestart: () => void;  // 重新挑战
  onBackToMenu: () => void; // 回到标题页
}
```
