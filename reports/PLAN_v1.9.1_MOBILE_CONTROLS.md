# 📱 v1.9.1 虚拟按键优化计划

**创建日期**: 2026-03-11

**版本**: v1.9.1

---

## 🐛 问题背景

### 当前状态
- v1.9.0 移除了虚拟按键，改用纯触摸手势控制
- 游戏画布尺寸：240×480px（移动端）
- 触摸手势：滑动移动、单击旋转、双击硬降

### 用户反馈
部分用户反馈纯触摸手势不够直观，希望能够有虚拟按键选项。

---

## 🎯 需求分析

### 核心需求
1. **可选虚拟按键** - 用户可以选择使用虚拟按键或触摸手势
2. **紧凑布局** - 虚拟按键不能占用太多屏幕空间
3. **可配置** - 用户可以开关虚拟按键、调整大小/透明度

### 功能范围
- ✅ 虚拟按键组件（精简版）
- ✅ 设置面板（开关/大小/透明度）
- ✅ 与触摸手势共存（可切换）
- ✅ 设置持久化（localStorage）

### 不在范围内
- ❌ 复杂布局模式（横屏/浮动）
- ❌ 触觉反馈（可选优化）
- ❌ 按键连发（可选优化）

## 📝 技术设计

### 组件结构
```
App.tsx
├── ResponsiveLayout
│   ├── 主菜单内容
│   └── 游戏区域
│       ├── GameCanvas（触摸手势）
│       ├── GameInfo
│       └── MobileControls（可选）
└── SettingsPanel
    └── 虚拟按键开关
```

### 布局方案
**固定底部紧凑版**:
```
┌─────────────────┐
│   游戏画面       │
│   240×480       │
│   [分数] [消除]  │
├─────────────────┤
│ [⬅️] [🔄] [➡️]  │ ← 精简按键
│ [⬇软] [💥硬]    │
└─────────────────┘
```

### 设置存储
```typescript
interface MobileControlsSettings {
  enabled: boolean;      // 是否启用
  size: 'small' | 'medium' | 'large';  // 尺寸
  opacity: number;       // 透明度 0.5-1.0
}
```
