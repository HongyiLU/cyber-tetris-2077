# 🏠 多助手聊天室

一个功能完整的多助手聊天界面，让千束、彩虹、代码、管家都能在同一个聊天室里聊天、协作、交互！

---

## ✨ 功能特点

### 核心功能
- 👥 **多助手同屏聊天** - 4个专业助手在同一个聊天室
- 🎯 **@特定助手** - 可以@千束、彩虹、代码、管家
- 💾 **本地存储** - 聊天记录自动保存
- ⏱️ **时间戳** - 每条消息有时间显示
- 🎨 **精美界面** - 深色主题，不同助手不同颜色
- 🎭 **正在输入提示** - 模拟真实对话体验

### 聊天室成员
| 头像 | 名字 | 角色 | 颜色 |
|------|------|------|------|
| 🎮 | 千束 | 首席游戏设计师 | 橙色 |
| 🎨 | 彩虹 | 首席艺术设计师 | 粉色 |
| 💻 | 代码 | 高级程序员 | 蓝色 |
| 📊 | 管家 | 项目经理 | 绿色 |
| 👤 | 用户 | 项目发起人 | 紫色 |

## 🚀 快速开始

### 方式1：直接在浏览器打开
1. 进入项目目录：`multi-agent-chatroom/`
2. 双击打开 `index.html`
3. 开始聊天！

### 方式2：用本地服务器（推荐）
```bash
cd multi-agent-chatroom
# 使用Python
python -m http.server 8080
# 或使用Node.js
npx serve .
# 然后在浏览器打开 http://localhost:8080
```

## 🎯 使用说明

### 发送消息
1. 在输入框输入你的消息
2. （可选）从下拉菜单选择要@的助手
3. 点击"📤"按钮或按回车键发送

### @特定助手
- 从下拉菜单选择要@的助手
- 该助手会收到你的消息并回复

### 清空聊天记录
- 点击右上角的"⚙️"按钮
- 确认清空
