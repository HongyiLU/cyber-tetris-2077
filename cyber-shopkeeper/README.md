# 🌃 赛博朋克掌柜 2077

一款结合《我是大掌柜》玩法的赛博朋克风格商店经营游戏！

## 🎮 游戏特色

### 赛博朋克视觉风格
- ✅ **霓虹色调** - 粉、青、黄、紫的经典赛博朋克配色
- ✅ **发光效果** - 霓虹灯、全息广告、脉冲效果
- ✅ **像素风格** - 像素风 + 赛博朋克的完美结合
- ✅ **未来街道** - 飞行汽车、全息广告、仿生人
- ✅ **建筑设计** - 霓虹招牌、发光建筑、高科技店面

### 核心玩法（类似《我是大掌柜》）
- ✅ **多种商店** - 5种不同类型的商店
- ✅ **商店解锁** - 达到等级解锁新商店
- ✅ **商店升级** - 提升等级，增加收益
- ✅ **员工系统** - 招聘、管理、分配员工
- ✅ **每日结算** - 计算收益，扣除工资
- ✅ **等级声望** - 升级、解锁、获得声望
- ✅ **商品系统** - 每个商店有独特商品
- ✅ **自动经营** - 安排员工后自动产生收益

## 🏪 商店类型

1. **🍜 义体拉面** - 赛博朋克世界最受欢迎的快餐（初始解锁）
2. **💊 义体诊所** - 维护和升级义体的专业诊所（初始解锁）
3. **🔫 武器商店** - 赛博朋克世界的武器供应商（5级解锁）
4. **🍸 霓虹酒吧** - 赛博朋克风格的夜店酒吧（8级解锁）
5. **👕 光污染时装** - 赛博朋克时尚服装品牌（12级解锁）

## 👥 员工类型

1. **🤖 基础仿生人** - 最基础的仿生人员工
2. **⚡ 高级仿生人** - 升级后的仿生人，效率更高
3. **💻 网络跑手** - 擅长市场和数据分析
4. **🔫 独行者** - 能处理各种麻烦事
5. **🕶️ 中间人** - 人脉广，能搞到稀缺资源
6. **🧠 AI经理** - 超级AI管理，全方位提升

## 🎯 如何游戏

### 基本操作
1. **点击建筑** - 打开商店详情界面
2. **管理商店** - 升级商店、查看收益
3. **招聘员工** - 在员工面板管理（开发中）
4. **下一天** - 点击"下一天"结算每日收益
5. **升级解锁** - 达到等级自动解锁新商店

### 游戏循环
1. 管理你的初始商店（拉面 + 诊所）
2. 招聘员工分配到商店
3. 每天获得收益，扣除工资
4. 升级商店，提升收益
5. 达到等级，解锁新商店
6. 扩大你的商业帝国！

## 📊 游戏系统

### 经济系统
- **₵ 现金** - 用于解锁、升级、招聘
- **⭐ 声望** - 经营获得，提升等级
- **📈 等级** - 解锁新商店和功能

### 商店系统
- **解锁** - 达到等级 + 支付费用
- **升级** - 支付费用，提升等级和收益
- **收益** - 基础收益 × 等级 × 员工加成
- **员工** - 分配员工增加效率

### 员工系统
- **招聘** - 一次性招聘费用
- **工资** - 每日从收益中扣除
- **效率** - 影响商店收益
- **魅力** - 影响顾客和声望

## 🎨 技术特点

- 纯HTML5 + CSS3 + JavaScript
- 赛博朋克像素风格
- LocalStorage数据保存
- 无需依赖，直接运行
- 响应式设计
