## 🔧 技术实现细节

### 1. 敌人类型数据结构

```typescript
interface EnemyType {
  id: string;              // 唯一标识
  name: string;            // 名称
  emoji: string;           // 表情符号/头像
  hp: number;              // 血量
  attackInterval: number;  // 攻击间隔（毫秒）
  attackDamage: number;    // 攻击伤害
  garbageRows: number;     // 垃圾行数
  description: string;     // 描述
  rarity: string;          // 稀有度
}
```

### 2. GameEngine 集成

**修改前**:
```typescript
public initBattle(enemyHp: number = 200): void {
  this.playerHp = 100;
  this.enemyHp = enemyHp;
  this.enemyMaxHp = enemyHp;
  this.battleState = BattleState.FIGHTING;
}
```

**修改后**:
```typescript
public initBattle(enemyType: string = 'slime'): void {
  const enemy = getEnemyType(enemyType);
  
  if (enemy) {
    this.currentEnemyType = enemy;
    this.playerHp = 100;
    this.playerMaxHp = 100;
    this.enemyHp = enemy.hp;
    this.enemyMaxHp = enemy.hp;
    this.enemyAttackInterval = enemy.attackInterval;
    this.battleState = BattleState.FIGHTING;
  }
}
```

### 3. 敌人攻击逻辑

**修改前**:
```typescript
private executeEnemyAttack(): void {
  this.addGarbageRow(1);  // 固定 1 行
  this.takeDamage(10);    // 固定 10 伤害
}
```

**修改后**:
```typescript
private executeEnemyAttack(): void {
  if (!this.currentEnemyType) {
    // 默认攻击（史莱姆）
    this.addGarbageRow(1);
    this.takeDamage(10);
  } else {
    // 根据敌人类型攻击
    this.addGarbageRow(this.currentEnemyType.garbageRows);
    this.takeDamage(this.currentEnemyType.attackDamage);
  }
}
```

## 📊 游戏策略影响

### 敌人难度对比

| 敌人 | 血量 | DPS | 难度 | 推荐等级 |
|------|------|-----|------|----------|
| 史莱姆 | 200 | 1.0 | ⭐ | 新手 |
| 哥布林 | 150 | 1.875 | ⭐⭐ | 入门 |
| 兽人 | 300 | 1.67 | ⭐⭐ | 进阶 |
| 幽灵 | 180 | 1.33 | ⭐⭐⭐ | 高手 |
| 巨龙 | 500 | 1.67 | ⭐⭐⭐⭐⭐ | 专家 |

**DPS 计算**: 伤害 ÷ (攻击间隔 ÷ 1000)

### 玩家策略建议

**对战史莱姆**:
- 适合新手练习
- 攻击间隔长（10 秒），有充足时间回复
- 垃圾行少（1 行），压力小

**对战哥布林**:
- 血量低但攻击频率高
- 需要快速消行，避免被连续攻击
- 垃圾行较多（2 行），注意棋盘管理

**对战兽人**:
- 血量高，需要持久战
- 攻击间隔长，但单次伤害高（20）
- 适合练习消行效率

**对战幽灵**:
- 攻击频率最快（6 秒）
- 单次伤害低（8），但累积伤害高
- 需要高度集中注意力

**对战巨龙**:
- Boss 级敌人，血量最高（500）
- 单次伤害最高（25），2-3 次攻击就可能失败
- 垃圾行最多（3 行），棋盘压力巨大
- 需要完美发挥才能获胜
