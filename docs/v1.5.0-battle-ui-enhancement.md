## 🔧 连击系统实现

### 核心逻辑

```typescript
// 连击时间窗口：5 秒
private comboTimeout: number = 5000;

// 在 lockPiece() 中更新连击
if (clearedLines > 0) {
  const currentTime = Date.now();
  if (currentTime - this.lastClearTime <= this.comboTimeout) {
    // 在连击时间内，增加连击数
    this.combo++;
  } else {
    // 超出连击时间，重置连击
    this.combo = 1;
  }
  this.lastClearTime = currentTime;
  
  // 更新最大连击
  if (this.combo > this.maxCombo) {
    this.maxCombo = this.combo;
  }
}
```

### 连击伤害加成

```typescript
// 战斗系统：消行时触发伤害
if (clearedLines > 0 && this.battleState === BattleState.FIGHTING) {
  // 计算连击加成
  const comboMultiplier = 1 + (this.combo - 1) * 0.1; // 每连击 +10% 伤害
  const baseDamage = this.calculateDamage(clearedLines);
  const damage = Math.floor(baseDamage * comboMultiplier);
  
  this.enemyTakeDamage(damage);
}
```

**伤害计算公式**:
```
最终伤害 = 基础伤害 × (1 + (连击数 - 1) × 0.1)
```

**示例**:
- 1 连击：基础伤害 × 1.0 (100%)
- 5 连击：基础伤害 × 1.4 (140%)
- 10 连击：基础伤害 × 1.9 (190%)
- 20 连击：基础伤害 × 2.8 (280%)

## 📊 游戏策略影响

### 连击策略

**保持连击的技巧**:
1. **快速消行** - 在 5 秒内完成下一次消行
2. **规划方块** - 提前规划方块位置，减少思考时间
3. **使用硬降** - 加快游戏节奏
4. **连击奖励** - 高连击造成显著伤害加成

**风险与回报**:
- **高风险**: 追求连击可能导致失误
- **高回报**: 20 连击时伤害接近 3 倍
- **平衡点**: 10-15 连击较为理想（1.9-2.4 倍伤害）

### 敌人对战策略

| 敌人 | 推荐连击策略 | 原因 |
|------|-------------|------|
| **史莱姆** | 稳健连击（5-10） | 攻击间隔长，有足够时间 |
| **哥布林** | 快速连击（10-15） | 攻击频率高，需要速战速决 |
| **兽人** | 持久连击（15-20） | 血量高，需要最大化伤害 |
| **幽灵** | 极限连击（20+） | 攻击最快，必须速胜 |
| **巨龙** | 完美连击（20+） | Boss 血量极高，需要最大输出 |

## 📝 使用示例

### 在 App.tsx 中集成

```tsx
import { DamageNumber } from './components/ui/DamageNumber';
import { ComboCounter } from './components/ui/ComboCounter';

function App() {
  const [gameState, setGameState] = useState<GameState | null>(null);
  const [damageNumbers, setDamageNumbers] = useState([]);

  // 显示伤害数字
  const showDamage = (value: number, type: DamageType, x: number, y: number) => {
    const id = Date.now();
    setDamageNumbers(prev => [...prev, { id, value, type, x, y }]);
  };

  return (
    <div className="game-container">
      {/* 连击计数器 */}
      {gameState && (
        <ComboCounter
          combo={gameState.combo}
          maxCombo={gameState.maxCombo}
          visible={gameState.battleState === BattleState.FIGHTING}
        />
      )}

      {/* 伤害数字 */}
      {damageNumbers.map(dn => (
        <DamageNumber
          key={dn.id}
          value={dn.value}
          type={dn.type}
          position={{ x: dn.x, y: dn.y }}
          onComplete={() => {
            setDamageNumbers(prev => prev.filter(d => d.id !== dn.id));
          }}
        />
      ))}

      {/* 游戏棋盘 */}
      <GameCanvas />
    </div>
  );
}
```

### 在 GameEngine 中使用

```typescript
// 创建游戏引擎
const engine = new GameEngine(10, 20);
```
