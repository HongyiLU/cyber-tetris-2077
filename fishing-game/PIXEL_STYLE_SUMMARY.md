## 🎨 像素风格设计要点

### 色彩方案
- 主色调: 复古像素蓝、木色、金色
- 阴影色: 深色调，制造层次感
- 高亮色: 浅色调，模拟像素高光

### 像素阴影技巧
- 外阴影: `4px 4px 0 #颜色` (硬边阴影)
- 内阴影: `inset -2px -2px 0 #深色, inset 2px 2px 0 #浅色`
- 多层阴影: 组合使用外阴影和内阴影

### 像素动画
- 使用 `step-end` 代替 `ease`，制造阶梯感
- 低帧率动画效果
- 位移以像素为单位 (8px, 16px, 32px...)

### 像素背景
- `repeating-linear-gradient` 创建条纹效果
- 8px或16px为一个周期
- 两种颜色交替
