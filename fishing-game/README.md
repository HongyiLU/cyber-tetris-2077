# 🎣 宁静钓鱼 (Serene Fishing)

一款轻松有趣的HTML5钓鱼游戏！

## 🎮 游戏特点

- **简单易懂的玩法:** 抛竿 → 等待 → 收线 → 收获！
- **多种鱼类:** 从普通的小鲫鱼到传说中的金龙鱼，7种鱼类等你来钓！
- **装备系统:** 升级你的钓鱼竿、鱼线和鱼饵，提高钓鱼效率！
- **进度系统:** 升级、收集图鉴、获得成就！
- **精美的视觉:** 手绘风格，流畅动画，轻松的游戏体验！

## 🎯 如何游戏

1. **抛竿:** 点击"抛竿"按钮开始蓄力，再次点击（或按空格键）完成抛竿
2. **等待:** 等待鱼儿上钩，观察浮漂动静
3. **收线:** 当鱼咬钩时，按住"收线"按钮，但要注意张力！
4. **收获:** 成功钓上鱼后，可以选择放入背包或放生获得双倍经验
5. **发展:** 出售鱼类获得金币，购买更好的装备，挑战更稀有的鱼！

## 🎛️ 控制说明

- **鼠标:** 点击按钮控制
- **空格键:** 抛竿/蓄力
- **E键:** 打开背包

## 🚀 开始游戏

直接用浏览器打开 `index.html` 即可开始游戏！

## 🎨 技术栈

- **HTML5:** 页面结构
- **CSS3:** 样式和动画
- **JavaScript (ES6+):** 游戏逻辑
- **LocalStorage:** 数据保存

## 📝 设计理念

这款游戏的设计目标是提供一个轻松、宁静的游戏体验。没有紧张的时间限制，没有复杂的操作，只需要享受钓鱼的过程。
