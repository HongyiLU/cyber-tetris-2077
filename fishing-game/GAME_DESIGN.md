# 宁静钓鱼 (Serene Fishing) - 游戏设计文档 🎣

## 1. 游戏概念

### 核心体验
- **类型:** 休闲模拟钓鱼游戏
- **氛围:** 宁静、放松、沉浸式
- **目标:** 享受钓鱼过程，收集鱼类，升级装备

### 核心玩法循环
1. 选择钓鱼位置
2. 抛竿
3. 等待咬钩
4. 收线钓鱼
5. 收获/放生
6. 购买/升级装备
7. 重复

---

## 2. 游戏系统设计

### 2.1 钓鱼机制

#### 抛竿系统
- 鼠标/触摸控制抛竿力度和方向
- 力度条显示抛投距离
- 落点影响鱼类种类

#### 咬钩系统
- 浮漂下沉动画表示咬钩
- 不同鱼类咬钩力度不同
- 时机窗口需要玩家反应

#### 收线系统
- 点击/按住收线
- 鱼会挣扎，需要控制张力
- 张力过大鱼线会断
- 张力过小鱼会逃脱

### 2.2 鱼类系统

#### 鱼类属性
```javascript
鱼类 = {
  name: 名称,
  rarity: 稀有度 (普通/ uncommon/ 稀有/ 史诗/ 传说),
  weightMin: 最小重量,
  weightMax: 最大重量,
  difficulty: 难度,
  behavior: 行为模式,
  habitat: 栖息地,
  time: 活跃时间,
  season: 活跃季节,
  description: 描述
}
```

#### 鱼类列表 (初始)
1. **小鲫鱼** - 普通，常见，容易钓
2. **鲤鱼** - 普通，中等大小
3. **草鱼** - uncommon，较大
4. **鲈鱼** - 稀有，有力的挣扎
5. **黑鱼** - 稀有，凶猛
6. **金鱼** - 史诗，观赏鱼
7. **金龙鱼** - 传说，极其稀有

### 2.3 装备系统

#### 钓鱼竿
- 初级竿: 基础，低价格
- 中级竿: 更好的控制，中等价格
- 高级竿: 优秀的控制，高价格
- 传说竿: 完美控制，极其昂贵

#### 鱼线
- 细线: 易断，不吓鱼
- 中线: 平衡
- 粗线: 坚韧，容易吓鱼

#### 鱼饵
- 面包虫: 普通鱼喜欢
- 蚯蚓: 大多数鱼喜欢
- 小鱼: 掠食性鱼喜欢
- 特制饵料: 稀有鱼喜欢

### 2.4 经济系统

#### 货币
- **金币:** 出售鱼类获得，购买装备
- **钓鱼点数:** 完成成就获得，解锁特殊物品

#### 鱼类价格
- 普通鱼: 5-20金币
- uncommon: 20-50金币
- 稀有: 50-150金币
- 史诗: 150-500金币
- 传说: 500+金币

### 2.5 进度系统

#### 等级
- 通过钓鱼获得经验值升级
- 升级解锁新装备、新地点
- 等级影响鱼类大小上限

#### 成就系统
- 收集特定鱼类
- 钓到特定重量
- 连续钓鱼
- 使用特定装备

#### 图鉴
- 记录已钓过的鱼类
- 显示鱼类信息和统计
- 收集奖励

## 4. 视觉风格

### 4.1 艺术风格
- 2D手绘风格
- 柔和的色彩
- 流畅的动画
- 简约但有细节

### 4.2 配色方案
- 主色: 蓝色系 (水、天空)
- 辅助色: 绿色系 (植物)
- 强调色: 橙色/黄色 (太阳、重点)

### 4.3 动画
- 水面波动
- 浮漂晃动
- 鱼线弯曲
- 鱼类游动
